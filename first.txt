Githubに作成したリモートリポジトリをローカルで管理するようにする方法
１．Githubにリモートリポジトリをつくり、sshのurlをコピーする。
２．ローカル（自分のPC）にリモートリポジトリと関連付けたいディレクトリを作成。
３．そのディレクトリに移動。
４．ターミナルで $ git clone <sshのurl>　としてEnter。
５．そのディレクトリの中に、リモートリポジトリと同名のディレクトリが作成され、その中に.gitという隠しディレクトリも作成される。これで完了
６．あとはこのディレクトリの中にプログラムコードなどを保存して、
$ git add .
$ git commit -m 'コメント'
$ git push origin
していけばOK。

$ git init はローカルだけで管理するときに使うと考えておけばいいかな。
